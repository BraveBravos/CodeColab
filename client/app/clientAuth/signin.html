Live Editing

CodeColab is a real-time collaborative suite made to increase productivity with teams and remote pairs. This application solves the issue of having access to the most recent code while eliminating the hassle of merging, rebasing, and submitting pull requests.

GitHub Integration

We integrated GitHub to eliminate merge-conflicts and rebasing altogether while maintaining the protection of existing code by implementing automated branch creation and keeping the basic GitHub functionality behind the scenes. We've also incorporated and simplified the commit and pull request process to be inclusive in our application, saving the functionality of GitHub and improving the user interaction.

Video Communication

Many developers work remotely and it can be a pain to access external means of communication. In the best case scenario it's a separate application, and we wanted to change that. Our video chat allows clear communication and the essence of teamwork to persist when dealing with remote users across the globe eliminating the need to seek outno external means of communication.
