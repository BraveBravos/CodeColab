Live Editing

Our goal in creating this real-time collaborative suite was to increase productivity and address the issue of having access to the most recent code. Without merging, rebasing, and submitting pull requests, we wanted to keep everyone up to date and simplify the process.

GitHub Integration

Working with teams, it can be difficult to code in the same files without the constant need to merge, rebase, and submit pull requests to keep everyone up to date. We integrated GitHub to eliminate merge-conflicts and rebasing altogether while maintaining the protection of existing code by implementing automated branch creation, keeping the basic GitHub functionality behind the scenes. We've also incorporated and simplified the commit and pull request process to be inclusive in our application, saving the functionality of GitHub and improving the user interaction.

Video Communication

Many developers work remotely and in a field where communication is very important, but can be a pain to access. Best case scenario, you're still in a separate application, and we wanted to change that. Our video chat allows clear communication and the essense of teamwork to persist when dealing with remote users across the globe without having to deal with finding external means of communication..
